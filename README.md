# Project Title: Social Network API

## Description:

his project is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This application uses MongoDB, Express.js, and Mongoose.


## Installation:
 - Javascript
 - Node.js
 - Express.js
 - MongoDB
 - Mongoose v 6.9.2

 ## Usage:

 
 To invoke the application, run npm start.
 When the server is started, the Mongoose models are synched to the MongoDB database.
 Open MongoDB and connect to the MongoDB URI mongodb://localhost:3000. On the list of databases, click on socialDB to see thoughts and users data.
 To create seed data and test the API routes, use Insomnia.
